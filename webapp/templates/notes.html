A Brief idea about Regex

A Regex (Regular Expression) is a sequence of characters which defines a pattern by using which we can search and manipulate the strings. Regex can be used in mostly all programming languages which can be used for managing data. In Python we can use Regex simply by importing 're' module.

Some of the functions available in the 're' module are,

re.match(): Checks if the beginning of a string matches the regex pattern.

re.findall(): Finds all non-overlapping matches of the regex pattern in the string and returns them as a list.

re.finditer(): Similar to findall(), but returns an iterator yielding match objects instead of a list.

re.IGNORECASE (or re.I): Makes the match case-insensitive.

re.MULTILINE (or re.M): Allows the start and end anchor (^ and $) to match at the start and end of each line, not just the start and end of the whole string
